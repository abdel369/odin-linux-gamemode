# Roadmap

## v0.1: safe SD-card layer

- Doctor script
- Apt upgrade guard
- Base package installer
- Device-access group setup
- FEX helper
- Gamescope helper
- Steam launch wrappers
- GameMode session file
- Compatibility report format

## v0.2: reproducible Steam/FEX modes

- Documented Steam x86-through-FEX path
- Documented Steam ARM64 path if available outside appliance distros
- Per-game profile system
- More compatibility reports
- Known-good Armbian image table

## v0.3: handheld polish

- Optional fake-suspend prototype
- Optional power-button handling
- Optional MangoHud presets
- Optional InputPlumber/HHD evaluation if native controls are not enough

## v0.4: image build layer

- Armbian userpatches/customize-image.sh integration
- Prebuilt image recipe
- First-boot setup wizard

## Not v0.x yet

- Internal install
- Android userdata resizing
- ABL flashing
- Automatic partition modification
